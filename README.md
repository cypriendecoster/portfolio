# Portfolio 2.0

Portfolio personnel construit avec React, Vite et Tailwind.

## Demarrage

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## A personnaliser

- Le texte du hero dans `src/App.jsx`
- Les projets affiches dans `src/App.jsx`
- Les liens de contact dans `src/App.jsx`
- Les couleurs et la direction visuelle dans `src/index.css`
